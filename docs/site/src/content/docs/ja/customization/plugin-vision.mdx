<Aside type="note">
Mutsumi には現在プラグインシステムがありません。このページは将来の構想を説明しています。
</Aside>

## なぜ今はないのか？

プラグインには以下が必要です：
- バージョン管理された API コントラクト
- プラグイン間のエラー分離
- 依存関係管理

現在の設計哲学：**コアが安定するまでミニマルに保つ。**

## 現在の「プラグイン」モデル

今日の Mutsumi の「プラグイン」は **JSON を書くあらゆるプログラム**です：

- AI Agent（Claude Code、Codex CLI、Gemini CLI）
- シェルスクリプト
- Cron ジョブ

`tasks.json` がユニバーサルインターフェースです。

## 将来の可能性（1.0 以降）

| コンセプト | 説明 |
|-----------|------|
| **カスタムビューウィジェット** | ランタイムで読み込まれるユーザー定義ウィジェット |
| **データトランスフォーマー** | レンダリング前にタスクを加工 |
| **通知統合** | Slack、Discord 連携 |
| **インポート/エクスポート** | Todoist、Notion 形式との変換 |
| **テンプレートタスク** | 定期タスクのテンプレート |

## タイムライン

プラグインシステムは **1.0 リリース後** に計画されています。
