## 内蔵テーマ

Mutsumi は 4 つのテーマを内蔵しています。`config.toml` で切り替え可能です。

```toml
theme = "monochrome-zen"
```

<CardGrid>
  <Card title="monochrome-zen（デフォルト）">
    ミニマルな白黒グレー。ライトシアンのアクセントカラー。最もクリーンで集中力を削がない。
  </Card>
  <Card title="nord">
    クールなトーン、控えめな配色。Nord カラーパレットに基づく。
  </Card>
  <Card title="dracula">
    高コントラスト、パープル寄り。人気の Dracula テーマに基づく。
  </Card>
  <Card title="solarized">
    暖かみのあるダーク配色。ティールのアクセント。
  </Card>
</CardGrid>

## テーマの構造

各テーマは以下のカラースロットで構成されます：

| スロット | 説明 |
|---|---|
| `background` | アプリ全体の背景色 |
| `surface` | パネルやカードの表面色 |
| `border` | ボーダーカラー |
| `text` | メインテキストカラー |
| `text_muted` | 控えめなテキストカラー |
| `accent` | アクセントカラー（ハイライト、フォーカス） |
| `error` | エラー表示カラー |
| `priority_high` | 高優先度タスクのカラー |
| `priority_normal` | 通常優先度タスクのカラー |
| `priority_low` | 低優先度タスクのカラー |

<Aside type="tip">
独自のテーマを作成したい場合は[カスタムテーマ](/ja/customization/custom-themes/)ページを参照してください。TOML ファイルでカスタムテーマを定義し、`~/.config/mutsumi/themes/` に配置できます。
</Aside>
