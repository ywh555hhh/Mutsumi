## 概要

Mutsumi は 3 つのキーバインドプリセットを提供します。`config.toml` で切り替え可能です。

```toml
keybindings = "vim"   # "vim" | "emacs" | "arrows"
```

マウス操作はすべてのプリセットで共通です。

## 共通バインディング（全プリセット共通）

以下のキーバインドはプリセットに関係なく常に有効です：

| キー | アクション |
|------|-----------|
| `Space` | 完了/未完了の切り替え |
| `Enter` | 詳細パネルの展開 |
| `Escape` | パネルを閉じる |
| `Tab` / `Shift+Tab` | タブの切り替え |
| `1` - `4` | タブのダイレクト切り替え |
| `n` | 新しいタスク |
| `e` | タスクの編集 |
| `i` | インライン編集 |
| `/` | 検索 |
| `+` / `=` | 優先度を上げる |
| `-` / `_` | 優先度を下げる |
| `y` | タスクをコピー |
| `p` | ペースト（下） |
| `P` | ペースト（上） |
| `z` | サブタスクの折りたたみ切り替え |
| `s` | ソート |
| `A` (`Shift+A`) | サブタスクの追加 |
| `?` | ヘルプ |

## マルチキーシーケンス

一部のアクションはマルチキーシーケンスで実行されます（KeyManager が処理）：

| シーケンス | アクション |
|---|---|
| `dd` | タスクの削除 |
| `gg` | リストの先頭にジャンプ |

</Aside>
