## 概要

Mutsumi は 3 つのキーバインドプリセットを提供します。`config.toml` で切り替え可能です。

```toml
keybindings = "vim"   # "vim" | "emacs" | "arrows"
```

マウス操作はすべてのプリセットで共通です。

## 共通バインディング（全プリセット共通）

以下のキーバインドはプリセットに関係なく常に有効です：

| キー | アクション |
|------|-----------|
| <kbd>Space</kbd> | 完了/未完了の切り替え |
| <kbd>Enter</kbd> | 詳細パネルの展開 |
| <kbd>Escape</kbd> | パネルを閉じる |
| <kbd>Tab</kbd> / <kbd>Shift+Tab</kbd> | タブの切り替え |
| `1` - `4` | タブのダイレクト切り替え |
| <kbd>n</kbd> | 新しいタスク |
| <kbd>e</kbd> | タスクの編集 |
| <kbd>i</kbd> | インライン編集 |
| <kbd>/</kbd> | 検索 |
| `+` / `=` | 優先度を上げる |
| `-` / `_` | 優先度を下げる |
| <kbd>y</kbd> | タスクをコピー |
| <kbd>p</kbd> | ペースト（下） |
| <kbd>P</kbd> | ペースト（上） |
| <kbd>z</kbd> | サブタスクの折りたたみ切り替え |
| <kbd>s</kbd> | ソート |
| <kbd>A</kbd> (<kbd>Shift+A</kbd>) | サブタスクの追加 |
| <kbd>?</kbd> | ヘルプ |

## マルチキーシーケンス

一部のアクションはマルチキーシーケンスで実行されます（KeyManager が処理）：

| シーケンス | アクション |
|---|---|
| <kbd>dd</kbd> | タスクの削除 |
| <kbd>gg</kbd> | リストの先頭にジャンプ |

<Aside type="tip">
キーバインドのカスタマイズについては[カスタムキーバインド](../../customization/custom-keybindings/)ページを参照してください。
</Aside>
