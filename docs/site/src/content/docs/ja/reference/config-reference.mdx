## 完全な設定テンプレート

```toml
# ── Mutsumi Configuration ──────────────────────────

# デフォルトタスクファイルパス（オプション）

# タスクリストに表示するカラム
columns = ["checkbox", "title", "tags", "priority"]

# 起動時のデフォルトタブ（マルチソース："main"、シングルソース：スコープ名）
default_tab = "main"

dashboard_max_tasks = 3

dashboard_show_completed = true
```

## key_overrides

個別のキーバインドを上書き：

```toml
[key_overrides]
quit = "ctrl+q"
cursor_down = "ctrl+j"
```

## `[[projects]]`

登録済みプロジェクトソース。各エントリには `name`（表示名）と `path`（`mutsumi.json` を含むディレクトリの絶対パス）があります。`mutsumi project add/remove` で管理します。

```toml
[[projects]]
name = "saas-app"
path = "/Users/you/projects/saas-app"

[[projects]]
name = "oshigrid"
path = "/Users/you/projects/oshigrid"
```

## 通知モード

| モード | 動作 |
|--------|------|
| `quiet` | 完全サイレント（デフォルト） |
| `badge` | 期限切れタスクをハイライト |
| `bell` | ターミナルベルを送信 |
| `system` | システム通知 |

## 設定コマンド

```bash
mutsumi config --edit     # $EDITOR で開く
mutsumi config --show     # 現在の設定を表示
mutsumi config --reset    # デフォルトにリセット
mutsumi config --path     # 設定ファイルパスを表示
```

```
└── keys/
```
