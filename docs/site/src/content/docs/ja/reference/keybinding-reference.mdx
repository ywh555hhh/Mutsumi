## 共通キーバインド（全プリセット共通）

| 操作 | キー | 説明 |
|------|------|------|
| 完了切替 | `Space` | タスクステータスを切替 |
| 詳細表示 | `Enter` | 詳細パネルを展開 |
| 閉じる | `Escape` | パネルを閉じる/キャンセル |
| 次のタブ | `Tab` | 次のスコープタブ |
| 前のタブ | `Shift+Tab` | 前のタブ |
| Today | `1` | Today タブへ |
| Week | `2` | Week タブへ |
| Month | `3` | Month タブへ |
| Inbox | `4` | Inbox タブへ |
| 新規作成 | `n` | 新規タスク作成 |
| 編集 | `e` | タスク編集 |
| インライン編集 | `i` | タイトルをインライン編集 |
| 検索 | `/` | 検索バーを開く |
| ソート | `s` | ソートオーバーレイ |
| 優先度上げ | `+` / `=` | 優先度を上げる |
| 優先度下げ | `-` / `_` | 優先度を下げる |
| コピー | `y` | タスクをコピー |
| 貼り付け | `p` | 下に貼り付け |
| サブタスク追加 | `A` | 子タスク追加 |
| 折りたたみ | `z` | サブタスク表示切替 |
| ヘルプ | `?` | ヘルプ画面表示 |

## ナビゲーションキー（プリセット別）

| 操作 | vim | emacs | arrows |
|------|-----|-------|--------|
| 下移動 | `j` | `Ctrl+N` | `↓` |
| 上移動 | `k` | `Ctrl+P` | `↑` |
| 先頭へ | `gg` | `Ctrl+A` | `Home` |
| 末尾へ | `G` | `Ctrl+E` | `End` |
| グループ折畳 | `h` | `Ctrl+B` | `←` |
| グループ展開 | `l` | `Ctrl+F` | `→` |
| タスク下移動 | `J` | `Ctrl+Shift+N` | `Shift+↓` |
| タスク上移動 | `K` | `Ctrl+Shift+P` | `Shift+↑` |
| 終了 | `q` | `Ctrl+Q` | `q` |

## マルチキーシーケンス

| シーケンス | 操作 |
|-----------|------|
| `dd` | タスク削除（確認あり） |
| `gg` | 先頭へジャンプ（vim） |

## マウス操作

| 操作 | 動作 |
|------|------|
| チェックボックスクリック | 完了状態を切替 |
| タスク行クリック | タスクを選択 |
| タイトルダブルクリック | インライン編集 |
| タブクリック | タブ切替 |
| `[+New]` クリック | 新規タスク作成 |
| スクロール | リストをスクロール |
