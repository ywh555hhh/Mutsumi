## 共通キーバインド（全プリセット共通）

| 操作 | キー | 説明 |
|------|------|------|
| 完了切替 | <kbd>Space</kbd> | タスクステータスを切替 |
| 詳細表示 | <kbd>Enter</kbd> | 詳細パネルを展開 |
| 閉じる | <kbd>Escape</kbd> | パネルを閉じる/キャンセル |
| 次のタブ | <kbd>Tab</kbd> | 次のスコープタブ |
| 前のタブ | <kbd>Shift+Tab</kbd> | 前のタブ |
| Today | `1` | Today タブへ |
| Week | `2` | Week タブへ |
| Month | `3` | Month タブへ |
| Inbox | `4` | Inbox タブへ |
| 新規作成 | <kbd>n</kbd> | 新規タスク作成 |
| 編集 | <kbd>e</kbd> | タスク編集 |
| インライン編集 | <kbd>i</kbd> | タイトルをインライン編集 |
| 検索 | <kbd>/</kbd> | 検索バーを開く |
| ソート | <kbd>s</kbd> | ソートオーバーレイ |
| 優先度上げ | `+` / `=` | 優先度を上げる |
| 優先度下げ | `-` / `_` | 優先度を下げる |
| コピー | <kbd>y</kbd> | タスクをコピー |
| 貼り付け | <kbd>p</kbd> | 下に貼り付け |
| サブタスク追加 | <kbd>A</kbd> | 子タスク追加 |
| 折りたたみ | <kbd>z</kbd> | サブタスク表示切替 |
| ヘルプ | <kbd>?</kbd> | ヘルプ画面表示 |

## ナビゲーションキー（プリセット別）

| 操作 | vim | emacs | arrows |
|------|-----|-------|--------|
| 下移動 | <kbd>j</kbd> | <kbd>Ctrl+N</kbd> | `↓` |
| 上移動 | <kbd>k</kbd> | <kbd>Ctrl+P</kbd> | `↑` |
| 先頭へ | <kbd>gg</kbd> | <kbd>Ctrl+A</kbd> | <kbd>Home</kbd> |
| 末尾へ | <kbd>G</kbd> | <kbd>Ctrl+E</kbd> | <kbd>End</kbd> |
| グループ折畳 | <kbd>h</kbd> | <kbd>Ctrl+B</kbd> | `←` |
| グループ展開 | <kbd>l</kbd> | <kbd>Ctrl+F</kbd> | `→` |
| タスク下移動 | <kbd>J</kbd> | <kbd>Ctrl+Shift+N</kbd> | <kbd>Shift+↓</kbd> |
| タスク上移動 | <kbd>K</kbd> | <kbd>Ctrl+Shift+P</kbd> | <kbd>Shift+↑</kbd> |
| 終了 | <kbd>q</kbd> | <kbd>Ctrl+Q</kbd> | <kbd>q</kbd> |

## マルチキーシーケンス

| シーケンス | 操作 |
|-----------|------|
| <kbd>dd</kbd> | タスク削除（確認あり） |
| <kbd>gg</kbd> | 先頭へジャンプ（vim） |

## マウス操作

| 操作 | 動作 |
|------|------|
| チェックボックスクリック | 完了状態を切替 |
| タスク行クリック | タスクを選択 |
| タイトルダブルクリック | インライン編集 |
| タブクリック | タブ切替 |
| `[+New]` クリック | 新規タスク作成 |
| スクロール | リストをスクロール |
