Mutsumi ships with 4 built-in themes. Switch via `config.toml` or `mutsumi init`.

## Monochrome Zen (Default)

Minimalist dark theme with mint accent. Inspired by Catppuccin Teal + Vercel's black-and-white aesthetic.

| Token | Hex | Usage |
|-------|-----|-------|
| `background` | `#0f0f0f` | Main background |
| `surface` | `#1a1a1a` | Card/panel background |
| `border` | `#333333` | Divider lines |
| `text` | `#e0e0e0` | Primary text |
| `text_muted` | `#666666` | Secondary text |
| `accent` | `#5de4c7` | Accent (mint green) |
| `error` | `#e06c75` | Error / high priority |
| `priority_high` | `#e06c75` | High priority |
| `priority_normal` | `#e5c07b` | Normal priority |
| `priority_low` | `#666666` | Low priority |

## Switch Theme

```toml
# ~/.config/mutsumi/config.toml
theme = "monochrome-zen"   # or "nord", "dracula", "solarized"
```

See [Custom Themes](/customization/custom-themes/) to create your own.
