Mutsumi ships with three keybinding presets. Mouse behavior is identical across all presets.

## Presets

<Tabs>
  <TabItem label="vim (default)">
    | Key | Action |
    |-----|--------|
    | <kbd>j</kbd> / <kbd>k</kbd> | Move down / up |
    | <kbd>G</kbd> | Jump to bottom |
    | <kbd>gg</kbd> | Jump to top (multi-key) |
    | <kbd>h</kbd> / <kbd>l</kbd> | Collapse / expand group |
    | <kbd>J</kbd> / <kbd>K</kbd> | Move task down / up |
    | <kbd>dd</kbd> | Delete task (multi-key) |
    | <kbd>q</kbd> | Quit |
  </TabItem>
  <TabItem label="emacs">
    | Key | Action |
    |-----|--------|
    | <kbd>Ctrl+N</kbd> / <kbd>Ctrl+P</kbd> | Move down / up |
    | <kbd>Ctrl+E</kbd> | Jump to bottom |
    | <kbd>Ctrl+A</kbd> | Jump to top |
    | <kbd>Ctrl+B</kbd> / <kbd>Ctrl+F</kbd> | Collapse / expand group |
    | <kbd>Ctrl+Shift+N</kbd> / <kbd>Ctrl+Shift+P</kbd> | Move task down / up |
    | <kbd>dd</kbd> | Delete task |
    | <kbd>Ctrl+Q</kbd> | Quit |
  </TabItem>
  <TabItem label="arrows">
    | Key | Action |
    |-----|--------|
    | `↓` / `↑` | Move down / up |
    | <kbd>End</kbd> | Jump to bottom |
    | <kbd>Home</kbd> | Jump to top |
    | `←` / `→` | Collapse / expand group |
    | <kbd>Shift+↓</kbd> / <kbd>Shift+↑</kbd> | Move task down / up |
    | <kbd>dd</kbd> | Delete task |
    | <kbd>q</kbd> | Quit |
  </TabItem>
</Tabs>

## Shared Bindings (All Presets)

| Key | Action |
|-----|--------|
| <kbd>Space</kbd> | Toggle done/pending |
| <kbd>Enter</kbd> | Show detail panel |
| <kbd>Escape</kbd> | Close detail / cancel |
| <kbd>Tab</kbd> / <kbd>Shift+Tab</kbd> | Next / previous tab |
| `1` `2` `3` `4` | Switch to Today / Week / Month / Inbox |
| <kbd>n</kbd> | New task |
| <kbd>e</kbd> | Edit task |
| <kbd>i</kbd> | Inline edit title |
| <kbd>/</kbd> | Search |
| <kbd>s</kbd> | Sort |
| `+` / `-` | Priority up / down |
| <kbd>y</kbd> / <kbd>p</kbd> | Copy / paste task |
| <kbd>A</kbd> (Shift+A) | Add subtask |
| <kbd>z</kbd> | Toggle fold |
| <kbd>?</kbd> | Show help screen |

## Multi-Key Sequences

Mutsumi supports multi-key sequences via `KeyManager`:

- <kbd>dd</kbd> — Delete selected task (with confirmation)
- <kbd>gg</kbd> — Jump to top of list (vim preset)

Type the keys in sequence — no timeout. Press <kbd>Escape</kbd> to cancel a partial sequence.

## Switch Preset

In `~/.config/mutsumi/config.toml`:

```toml
keybindings = "vim"     # or "emacs" or "arrows"
```

Or during initial setup:

```bash
mutsumi init
```
