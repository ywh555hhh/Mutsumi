## CRUD Operations

### Create Task

Trigger: <kbd>n</kbd> key or click `[+New]` button.

A dialog opens with fields:
- **Title** (required)
- **Scope**: Day / Week / Month / Inbox (defaults to current tab)
- **Priority**: High / Normal / Low (defaults to Normal)
- **Tags**: Comma-separated input

UUIDv7 ID and `created_at` are auto-generated.

### Edit Task

- Press <kbd>e</kbd> to open the edit form for the selected task
- Press <kbd>i</kbd> for inline title editing directly in the list row
- Double-click a title to inline edit
- Press <kbd>Enter</kbd> to confirm, <kbd>Escape</kbd> to cancel

### Delete Task

- Press <kbd>dd</kbd> (vim-style two-key sequence)
- A confirmation bar appears: `"Delete 'xxx'? [y/N]"`
- Press <kbd>y</kbd> to confirm, any other key to cancel
- The task and all its children are removed

### Toggle Status

- Press <kbd>Space</kbd> or click the checkbox
- `pending` → `done`: auto-fills `completed_at`
- `done` → `pending`: clears `completed_at`
- Immediate write-back, no confirmation needed

## Subtasks

Tasks can have nested subtasks via the `children` array:

```
[ ] Refactor Auth module          ***
  └─ [x] Install PyJWT            (1/2)
  └─ [ ] Write middleware
```

- Press <kbd>A</kbd> (Shift+A) to add a subtask to the selected task
- Press <kbd>z</kbd> to toggle fold (show/hide subtasks)
- Cascading completion: marking parent done marks all children done

## Priority Cycling

| Key | Action |
|-----|--------|
| `+` or `=` | Increase priority (low → normal → high) |
| `-` or `_` | Decrease priority (high → normal → low) |

Priority is clamped at boundaries — pressing `+` on a `high` task does nothing.

## Copy / Paste

| Key | Action |
|-----|--------|
| <kbd>y</kbd> | Copy selected task to internal clipboard |
| <kbd>p</kbd> | Paste task below current selection |
| <kbd>P</kbd> (Shift+P) | Paste task above current selection |

## Reorder Tasks

| Preset | Move Down | Move Up |
|--------|-----------|---------|
| vim | <kbd>J</kbd> (Shift+J) | <kbd>K</kbd> (Shift+K) |
| emacs | <kbd>Ctrl+Shift+N</kbd> | <kbd>Ctrl+Shift+P</kbd> |
| arrows | <kbd>Shift+↓</kbd> | <kbd>Shift+↑</kbd> |

## Search

Press <kbd>/</kbd> to open the search bar.

- Real-time filtering as you type
- Search scope: title + tags + description
- Non-matching tasks are **dimmed** (not hidden) — preserves spatial context
- <kbd>Escape</kbd> exits search and restores the full list

## Sort

Press <kbd>s</kbd> to open the sort overlay.

Sort fields: `title`, `priority`, `status`, `due`

Press <kbd>r</kbd> to toggle reverse order. Navigate with <kbd>h/l</kbd> or arrow keys.
