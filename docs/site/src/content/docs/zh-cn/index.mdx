你不缺专注力。你缺的是一张随时能瞟一眼的线程清单。

写着代码忽然被拉进群聊，回完消息发现刚才那个 bug 的思路断了——
这不是你的问题，这就是多线程工作的代价。Mutsumi 把你手头所有的事情
钉在屏幕角落：一个快捷键唤出来，再一个快捷键收回去，轻到你根本感觉不到她的存在。

```
┌─────────────────────────────────────────────────────┐
│  [* 主面板] [个人] [proj-a] [proj-b]     mutsumi ~  │
├─────────────────────────────────────────────────────┤
│  v 高优先 ───────────────────────────────────────   │

│  [ ] Review PR #42                   dev         ** │

│  6 tasks · 2 done · 4 pending                       │
└─────────────────────────────────────────────────────┘
```

## 怎么跑起来的

```mermaid
graph LR
    A["AI Agent<br/>(Controller)"] -->|"写入"| T["mutsumi.json<br/>(Model)"]
    T -->|"监视"| V["你 (TUI)<br/>(View)"]
    V -->|"交互"| T
```

Agent 写 JSON，Mutsumi 盯着文件变化自动刷新，你瞟一眼就行。想改就直接在 TUI 里点，改完自动写回去。
