## 概述

如果预设方案（vim / emacs / arrows）不能完全满足你的需求，可以通过 `key_overrides` 覆盖个别快捷键，而不必从头定义整套键位。

## 基本用法

在 `config.toml` 中添加 `key_overrides` 字段：

```toml
keybindings = "vim"

[key_overrides]
quit = "ctrl+q"
cursor_down = "ctrl+j"
cursor_up = "ctrl+k"
```

这会基于 vim 预设，但把退出键改为 <kbd>Ctrl+q</kbd>，导航键改为 <kbd>Ctrl+j</kbd>/<kbd>Ctrl+k</kbd>。

## 可覆盖的操作

以下是所有可覆盖的操作名称：

### 导航

| 操作名 | 默认键 (vim) | 说明 |
|---|---|---|
| `cursor_down` | <kbd>j</kbd> | 光标下移 |
| `cursor_up` | <kbd>k</kbd> | 光标上移 |
| `cursor_top` | <kbd>gg</kbd> | 跳到顶部 |
| `cursor_bottom` | <kbd>G</kbd> | 跳到底部 |
| `collapse_group` | <kbd>h</kbd> | 折叠分组 |
| `expand_group` | <kbd>l</kbd> | 展开分组 |

### 任务操作

| 操作名 | 默认键 | 说明 |
|---|---|---|
| `toggle_done` | <kbd>Space</kbd> | 切换完成状态 |
| `new_task` | <kbd>n</kbd> | 新建任务 |
| `edit_task` | <kbd>e</kbd> | 编辑任务 |
| `inline_edit` | <kbd>i</kbd> | 行内编辑标题 |
| `delete_task` | <kbd>dd</kbd> | 删除任务 |
| `show_detail` | <kbd>Enter</kbd> | 展开详情面板 |
| `close_detail` | <kbd>Escape</kbd> | 关闭详情面板 |

### 优先级与移动

| 操作名 | 默认键 | 说明 |
|---|---|---|
| `priority_up` | `+` / `=` | 提升优先级 |
| `priority_down` | `-` / `_` | 降低优先级 |
| `move_down` | <kbd>J</kbd> | 下移任务 |
| `move_up` | <kbd>K</kbd> | 上移任务 |
| `copy_task` | <kbd>y</kbd> | 复制任务 |
| `paste_task` | <kbd>p</kbd> | 粘贴任务 |
| `paste_task_above` | <kbd>P</kbd> | 在上方粘贴 |

### 视图切换

| 操作名 | 默认键 | 说明 |
|---|---|---|
| `next_tab` | <kbd>Tab</kbd> | 下一个标签页 |
| `prev_tab` | <kbd>Shift+Tab</kbd> | 上一个标签页 |
| `tab_1` | `1` | Today 标签 |
| `tab_2` | `2` | Week 标签 |
| `tab_3` | `3` | Month 标签 |
| `tab_4` | `4` | Inbox 标签 |

### 其他

| 操作名 | 默认键 | 说明 |
|---|---|---|
| `search` | <kbd>/</kbd> | 搜索 |
| `sort` | <kbd>s</kbd> | 排序 |
| `toggle_fold` | <kbd>z</kbd> | 切换折叠 |
| `add_child` | <kbd>A</kbd> | 添加子任务 |
| `show_help` | <kbd>?</kbd> | 显示帮助 |
| `quit` | <kbd>q</kbd> | 退出 |

## 键名格式

键名使用 Textual 的按键格式：

| 格式 | 示例 |
|---|---|
| 单键 | <kbd>j</kbd>, <kbd>k</kbd>, <kbd>n</kbd>, <kbd>q</kbd> |
| 修饰键 | <kbd>ctrl+q</kbd>, <kbd>shift+g</kbd>, <kbd>ctrl+shift+n</kbd> |
| 特殊键 | <kbd>space</kbd>, <kbd>enter</kbd>, <kbd>escape</kbd>, <kbd>tab</kbd> |
| 方向键 | <kbd>up</kbd>, <kbd>down</kbd>, <kbd>left</kbd>, <kbd>right</kbd> |
| 功能键 | <kbd>home</kbd>, <kbd>end</kbd>, <kbd>delete</kbd>, `insert` |

## 示例：混合方案

```toml
# 基于 vim 预设，但用 Ctrl+q 退出，用方向键导航
keybindings = "vim"

[key_overrides]
quit = "ctrl+q"
cursor_down = "down"
cursor_up = "up"
cursor_top = "home"
cursor_bottom = "end"
```

<Aside type="note">
`key_overrides` 只替换已有绑定的按键。如果指定的操作名在预设中不存在，覆盖会被忽略。
</Aside>

## 完全自定义键位

如果需要从头定义所有键位，可以创建自定义键位文件：

```toml
# ~/.config/mutsumi/keys/custom.toml

[keys]
move_down = "j"
move_up = "k"
toggle_done = "space"
new_task = "n"
edit_task = "e"
delete_task = "d,d"    # chord：两次按键
quit = "q"
search = "/"
help = "?"
tab_next = "tab"
tab_prev = "shift+tab"
tab_1 = "1"
tab_2 = "2"
tab_3 = "3"
tab_4 = "4"
expand = "enter"
top = "g"
bottom = "shift+g"
```
