## 概述

Mutsumi 提供三种键位预设，在配置中一键切换。所有预设共享相同的鼠标行为和功能键（Space、Enter、Escape、Tab 等），只有导航键和退出键不同。

## 通用快捷键（所有预设共享）

以下快捷键在所有预设中完全相同：

| 按键 | 操作 |
|---|---|
| <kbd>Space</kbd> | 切换任务完成状态 |
| <kbd>Enter</kbd> | 展开/折叠详情面板 |
| <kbd>Escape</kbd> | 关闭详情面板 |
| <kbd>Tab</kbd> | 切换到下一个标签页 |
| <kbd>Shift+Tab</kbd> | 切换到上一个标签页 |
| `1` / `2` / `3` / `4` | 快速切换到第 N 个标签页 |
| <kbd>n</kbd> | 新建任务 |
| <kbd>e</kbd> | 编辑任务 |
| <kbd>i</kbd> | 行内编辑标题 |
| <kbd>/</kbd> | 搜索任务 |
| <kbd>dd</kbd> | 删除任务（需确认） |
| `+` / `=` | 提升优先级 |
| `-` / `_` | 降低优先级 |
| <kbd>y</kbd> | 复制任务 |
| <kbd>p</kbd> | 粘贴任务（下方） |
| <kbd>P</kbd> (Shift+p) | 粘贴任务（上方） |
| <kbd>z</kbd> | 切换子任务折叠 |
| <kbd>s</kbd> | 排序 |
| <kbd>A</kbd> (Shift+a) | 添加子任务 |
| <kbd>?</kbd> | 显示帮助 |

<Aside type="note">
<kbd>dd</kbd> 和 <kbd>gg</kbd> 是多键序列（chord），由 KeyManager 处理。按第一个键后需在短时间内按第二个键。
</Aside>

## 切换预设

在 `~/.config/mutsumi/config.toml` 中设置：

```toml
keybindings = "vim"     # "vim" | "emacs" | "arrows"
```

或使用 CLI：

```bash
mutsumi init --defaults   # 然后编辑 config.toml
mutsumi config --edit     # 直接打开编辑器
```

## 自定义快捷键

如果预设不能满足需求，可以通过 `key_overrides` 覆盖个别快捷键。详见 [自定义快捷键](../../customization/custom-keybindings/)。
