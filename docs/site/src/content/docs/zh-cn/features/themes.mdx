## 内置主题

Mutsumi 附带 4 款精心设计的暗色主题。

### Monochrome Zen（默认）

极简黑白灰，点缀色为淡青绿。

| 颜色变量 | 色值 | 说明 |
|---|---|---|
| background | `#0f0f0f` | 深黑背景 |
| surface | `#1a1a1a` | 卡片/面板背景 |
| border | `#333333` | 边框 |
| text | `#e0e0e0` | 主文字 |
| text_muted | `#666666` | 次要文字 |
| accent | `#5de4c7` | 点缀色（青绿） |
| error | `#e06c75` | 错误色 |
| priority_high | `#e06c75` | 高优先级 |
| priority_normal | `#e5c07b` | 普通优先级 |
| priority_low | `#666666` | 低优先级 |

## 切换主题

在 `~/.config/mutsumi/config.toml` 中设置：

```toml
theme = "monochrome-zen"   # "monochrome-zen" | "nord" | "dracula" | "solarized"
```

## 主题对照表

| 主题 | 风格 | 点缀色 |
|---|---|---|
| `monochrome-zen` | 极简黑白，青绿点缀（默认） | `#5de4c7` |
| `nord` | 冷色调，克制 | `#88c0d0` |
| `dracula` | 高对比度，紫色调 | `#bd93f9` |
| `solarized` | 暖色暗底，青蓝点缀 | `#2aa198` |

</Aside>

## 颜色变量说明

每个主题包含以下颜色变量：

| 变量 | 用途 |
|---|---|
| `background` | 整体背景色 |
| `surface` | 卡片、面板的背景色 |
| `border` | 边框颜色 |
| `text` | 主文字颜色 |
| `text_muted` | 次要/灰色文字 |
| `accent` | 品牌/交互点缀色 |
| `error` | 错误提示色 |
| `priority_high` | 高优先级任务颜色 |
| `priority_normal` | 普通优先级任务颜色 |
| `priority_low` | 低优先级任务颜色 |
