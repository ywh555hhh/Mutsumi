## 完整配置模板

## key_overrides

覆盖单个快捷键：

```toml
[key_overrides]
quit = "ctrl+q"
cursor_down = "ctrl+j"
```

## `[[projects]]`

注册的项目来源。每个条目包含 `name`（显示名称）和 `path`（包含 `mutsumi.json` 的目录的绝对路径）。通过 `mutsumi project add/remove` 管理。

```toml
[[projects]]
name = "saas-app"
path = "/Users/you/projects/saas-app"

[[projects]]
name = "oshigrid"
path = "/Users/you/projects/oshigrid"
```

## 通知模式

| 模式 | 行为 |
|------|------|
| `quiet` | 完全静默（默认） |
| `badge` | 过期任务在 TUI 中高亮 |
| `bell` | 发送终端响铃 |
| `system` | 系统通知 |

## 配置命令

```bash
mutsumi config --edit     # 用 $EDITOR 编辑
mutsumi config --show     # 打印当前配置
mutsumi config --reset    # 重置为默认值
mutsumi config --path     # 打印配置文件路径
```
