## 共享快捷键（所有预设通用）

| 操作 | 按键 | 说明 |
|------|------|------|
| 切换完成 | <kbd>Space</kbd> | 切换任务状态 |
| 查看详情 | <kbd>Enter</kbd> | 展开详情面板 |
| 关闭 | <kbd>Escape</kbd> | 关闭面板/取消 |
| 下一标签 | <kbd>Tab</kbd> | 切换下一个标签页 |
| 上一标签 | <kbd>Shift+Tab</kbd> | 切换上一个标签页 |
| 今日 | `1` | 切换到 Today |
| 本周 | `2` | 切换到 Week |
| 本月 | `3` | 切换到 Month |
| 收件箱 | `4` | 切换到 Inbox |
| 新建 | <kbd>n</kbd> | 新建任务 |
| 编辑 | <kbd>e</kbd> | 编辑任务 |
| 行内编辑 | <kbd>i</kbd> | 行内编辑标题 |
| 搜索 | <kbd>/</kbd> | 打开搜索栏 |
| 排序 | <kbd>s</kbd> | 打开排序面板 |
| 优先级升 | `+` 或 `=` | 提升优先级 |
| 优先级降 | `-` 或 `_` | 降低优先级 |
| 复制 | <kbd>y</kbd> | 复制任务 |
| 粘贴 | <kbd>p</kbd> | 在下方粘贴 |
| 添加子任务 | <kbd>A</kbd> | 添加子任务 |
| 折叠/展开 | <kbd>z</kbd> | 切换子任务显示 |
| 帮助 | <kbd>?</kbd> | 显示帮助 |

## 导航键（按预设区分）

| 操作 | vim | emacs | arrows |
|------|-----|-------|--------|
| 下移 | <kbd>j</kbd> | <kbd>Ctrl+N</kbd> | `↓` |
| 上移 | <kbd>k</kbd> | <kbd>Ctrl+P</kbd> | `↑` |
| 跳到顶部 | <kbd>gg</kbd> | <kbd>Ctrl+A</kbd> | <kbd>Home</kbd> |
| 跳到底部 | <kbd>G</kbd> | <kbd>Ctrl+E</kbd> | <kbd>End</kbd> |
| 折叠分组 | <kbd>h</kbd> | <kbd>Ctrl+B</kbd> | `←` |
| 展开分组 | <kbd>l</kbd> | <kbd>Ctrl+F</kbd> | `→` |
| 下移任务 | <kbd>J</kbd> | <kbd>Ctrl+Shift+N</kbd> | <kbd>Shift+↓</kbd> |
| 上移任务 | <kbd>K</kbd> | <kbd>Ctrl+Shift+P</kbd> | <kbd>Shift+↑</kbd> |
| 退出 | <kbd>q</kbd> | <kbd>Ctrl+Q</kbd> | <kbd>q</kbd> |

## 多键序列

| 序列 | 操作 |
|------|------|
| <kbd>dd</kbd> | 删除任务（需确认） |
| <kbd>gg</kbd> | 跳到顶部（vim 预设） |

## 鼠标操作

| 操作 | 行为 |
|------|------|
| 点击复选框 | 切换完成状态 |
| 点击任务行 | 选中任务 |
| 双击标题 | 行内编辑 |
| 点击标签页 | 切换标签 |
| 点击 `[+New]` | 新建任务 |
| 滚轮 | 滚动列表 |
