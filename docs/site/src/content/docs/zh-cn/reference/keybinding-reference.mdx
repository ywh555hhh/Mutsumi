## 共享快捷键（所有预设通用）

| 操作 | 按键 | 说明 |
|------|------|------|
| 切换完成 | `Space` | 切换任务状态 |
| 查看详情 | `Enter` | 展开详情面板 |
| 关闭 | `Escape` | 关闭面板/取消 |
| 下一标签 | `Tab` | 切换下一个标签页 |
| 上一标签 | `Shift+Tab` | 切换上一个标签页 |
| 今日 | `1` | 切换到 Today |
| 本周 | `2` | 切换到 Week |
| 本月 | `3` | 切换到 Month |
| 收件箱 | `4` | 切换到 Inbox |
| 新建 | `n` | 新建任务 |
| 编辑 | `e` | 编辑任务 |
| 行内编辑 | `i` | 行内编辑标题 |
| 搜索 | `/` | 打开搜索栏 |
| 排序 | `s` | 打开排序面板 |
| 优先级升 | `+` 或 `=` | 提升优先级 |
| 优先级降 | `-` 或 `_` | 降低优先级 |
| 复制 | `y` | 复制任务 |
| 粘贴 | `p` | 在下方粘贴 |
| 添加子任务 | `A` | 添加子任务 |
| 折叠/展开 | `z` | 切换子任务显示 |
| 帮助 | `?` | 显示帮助 |

## 导航键（按预设区分）

| 操作 | vim | emacs | arrows |
|------|-----|-------|--------|
| 下移 | `j` | `Ctrl+N` | `↓` |
| 上移 | `k` | `Ctrl+P` | `↑` |
| 跳到顶部 | `gg` | `Ctrl+A` | `Home` |
| 跳到底部 | `G` | `Ctrl+E` | `End` |
| 折叠分组 | `h` | `Ctrl+B` | `←` |
| 展开分组 | `l` | `Ctrl+F` | `→` |
| 下移任务 | `J` | `Ctrl+Shift+N` | `Shift+↓` |
| 上移任务 | `K` | `Ctrl+Shift+P` | `Shift+↑` |
| 退出 | `q` | `Ctrl+Q` | `q` |

## 多键序列

| 序列 | 操作 |
|------|------|
| `dd` | 删除任务（需确认） |
| `gg` | 跳到顶部（vim 预设） |

## 鼠标操作

| 操作 | 行为 |
|------|------|
| 点击复选框 | 切换完成状态 |
| 点击任务行 | 选中任务 |
| 双击标题 | 行内编辑 |
| 点击标签页 | 切换标签 |
| 点击 `[+New]` | 新建任务 |
| 滚轮 | 滚动列表 |
